# Lightweight Remote Administration Tool (RAT)

Минималистичный кроссплатформенный инструмент на чистом C++ для удаленного администрирования и выполнения системных команд по архитектуре Клиент-Сервер. Проект разработан в учебных целях для изучения низкоуровневого сетевого программирования, управления процессами в Linux и работы с POSIX Sockets.

## ✨ Особенности проекта
* **Архитектура Клиент-Сервер:** Серверная часть принимает входящие соединения и предоставляет интерактивный интерфейс управления (`CLI`), а клиентская часть (агент) исполняет поступающие инструкции.
* **Многопоточность:** Сервер использует `std::thread` для асинхронной обработки подключений, позволяя изолировать сессии.
* **Демонизация (Background Execution):** Реализован классический алгоритм перевода процесса агента в фоновый режим (двойной `fork()`, создание новой сессии через `setsid()`, перенаправление стандартных потоков ввода/вывода в `/dev/null`), что позволяет утилите работать скрытно от терминала.
* **Межпроцессное взаимодействие (IPC):** Агент использует функцию `popen()` для безопасного выполнения системных команд оболочки и динамического чтения их вывода (stdout/stderr) с последующей отправкой результата обратно на сервер.
* **Очистка и валидация трафика:** На стороне агента реализована фильтрация сетевых буферов от служебных символов переноса строк (`\n`, `\r`), что гарантирует корректную интерпретацию команд командной строкой.

## 🛠️ Стек технологий
* **Язык программирования:** C++ (стандарт C++11 и выше)
* **Сетевой стек:** POSIX Sockets (`sys/socket.h`, `netinet/in.h`)
* **Системное API:** Linux POSIX API (`unistd.h`, `sys/stat.h`, `fcntl.h`)
